# Countdown Timer

## ⚙️ Features

<p> ✅  Countdown Timer </p>
<p> ✅  Pause and Resume </p>
<p> ✅  Speed Selector </p>
<p> ✅  Dark theme </p>
<p> ✅  Datetime by timezone </p>

<br>
